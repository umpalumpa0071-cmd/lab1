# задание 1
## задача 2
### текст задачи
Создать список из n одинаковых элементов (элемент вводится)
### Алгоритм решения
1) запрашиваем у пользователя значение элемента и количество его повторений(n)
2) проверяем условие n>0,
если не удовлетворяет условию то выводим ошибку
4) создаем список при помощи List.replicate n x,
где n это количество повторений числа, а x само число
6) выводим на экран итоговый список

# задание 2
## задача 2
### текст задачи
Найти произведение цифр натурального числа
### алгоритм решения
1) вводим число
2) с помощью рекурсии находим произведение всех цифр числа
умножаем первую цифру, убираем ее, и умножаем на первую цифру нового числа,
и так проходим до конца, пока не останется последняя цифра
4) выводим
